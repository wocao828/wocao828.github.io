hexo github搭建个人博客


2018-12-3

前言:在Windows下使用hexo github搭建个人的博客

安装配置node.js

安装配置git

注册一个github账号，新建一个项目，格式为项目名.github.io

创建一个文件叫index.html,写入内容

在e盘创建一个blog文件夹

输入npm install -g hexo-cli安装hexo

输入hexo出现如下

初始化hexo

输入hexo generate

将blog用vscode打开，更改配置文件_config.yml中的title,author,language,url等配置项

输入hexo server

在浏览器中打开localhost:4000出现

更换一个主题:在theme下用git下载一个主题，git clone https://github.com/shuiRong/hexo-theme-Ada如下

更改配置文件_config.yml下的theme

输入hexo clean

输入hexo generate


输入hexo server


重新打开localhost:4000


安装配置githubdesktop,输入github账号,然后create repository，输入项目名，

将blog下public的所有文件复制到文档的github下面的项目名下面

然后在githubdesktop下的summary和description写入东西,点commit to master

然后点fetch origin同步上去，最后输入https://项目名,出现如下
